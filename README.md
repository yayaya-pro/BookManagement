### 对于后端返回数据有使用mock.js本地返回假数据，有向后端请求数据库测试，也有向easy-mock调用接口接收数据
### 项目主要使用vue，element-plus框架，向后端请求数据使用axios，并用proxy解决跨域问题
### 项目亮点：利用路由守卫来更改标签页标题；使用路由懒加载提高性能
